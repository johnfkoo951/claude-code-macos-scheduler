# 사용법 / Usage Guide

## 앱 실행

앱을 실행하면 3-column 레이아웃의 메인 윈도우가 표시됩니다.
동시에 메뉴 바에 달력 아이콘이 추가됩니다.

---

## Job 관리

### Job 생성

1. 툴바의 **+** 버튼 클릭
2. 오른쪽 디테일 패널에서 편집:

| 필드 | 설명 |
|------|------|
| **이름** | Job을 구분할 이름 |
| **폴더** | 분류용 폴더 선택 |
| **프롬프트** | Claude에게 전달할 프롬프트 |
| **스케줄** | 실행 주기 설정 |
| **활성화** | ON이면 launchd에 등록되어 자동 실행 |
| **백그라운드** | ON이면 화면 없이 실행, OFF면 iTerm에서 실행 |

모든 변경사항은 **자동 저장**됩니다.

### Job 삭제

- 디테일 패널 하단의 **삭제** 버튼
- 또는 Job 목록에서 우클릭 > **삭제**

### Job 즉시 실행

- 디테일 패널의 **즉시 실행** 버튼
- 또는 Job 목록에서 **더블클릭**
- 또는 메뉴 바 아이콘에서 실행

## 스케줄 설정

### 간격 모드 (Interval)

매 N초/분/시간마다 반복 실행합니다.

| 설정 | 설명 |
|------|------|
| 값 | 숫자 입력 (Stepper) |
| 단위 | 초 / 분 / 시간 |

예시: `30분` = 매 30분마다 실행

### 매일 모드 (Daily)

매일 지정된 시간에 실행합니다.

| 설정 | 설명 |
|------|------|
| 시간 | DatePicker (시:분) |

예시: `09:00` = 매일 오전 9시

### 매주 모드 (Weekly)

매주 지정된 요일과 시간에 실행합니다.

| 설정 | 설명 |
|------|------|
| 요일 | Picker 또는 원형 버튼으로 선택 |
| 시간 | DatePicker (시:분) |

예시: `월요일 09:00` = 매주 월요일 오전 9시

> **참고**: 스케줄은 macOS launchd를 통해 관리됩니다.
> 앱이 꺼져 있어도 Mac이 켜져 있으면 예약된 시간에 실행됩니다.

## 실행 모드

### 백그라운드 모드

- Claude CLI를 화면 없이 실행
- `--dangerously-skip-permissions -p "<프롬프트>"` 플래그 사용
- 결과는 로그 파일에 저장
- 스케줄 자동 실행에 적합

### iTerm 모드

- iTerm에 새 탭을 열어 Claude CLI 실행
- AppleScript로 자동화
- 실행 과정을 실시간으로 확인 가능
- 디버깅이나 모니터링에 적합

> **요구사항**: iTerm 앱이 설치되어 있어야 합니다.
> 첫 실행 시 자동화 접근 권한을 허용해야 합니다.

## 폴더 관리

### 폴더 생성
사이드바 하단의 **+** 버튼

### 폴더 이름 변경
폴더를 **더블클릭**하거나 우클릭 > **이름 변경**

### 폴더 삭제
사이드바 하단의 **-** 버튼 또는 우클릭 > **삭제**

> **참고**: "Default" 폴더는 삭제하거나 이름을 변경할 수 없습니다.

### 폴더 색상
폴더 생성 시 9가지 프리셋 색상 중 선택:
Blue, Purple, Pink, Red, Orange, Yellow, Green, Teal, Gray

## 메뉴 바

메뉴 바의 달력 아이콘(📅)을 클릭하면:

- **활성화된 Job 목록** 표시
- 각 Job의 **즉시 실행** 버튼
- **앱 열기** — 메인 윈도우 표시
- **종료** — 앱 완전 종료

## 로그 확인

### 로그 저장 위치
```
~/Library/Application Support/ClaudeScheduler/Logs/<job-uuid>/
```

### 로그 파일
| 파일 | 설명 |
|------|------|
| `latest.log` | 모든 실행 결과가 누적 |
| `YYYY-MM-DD_HHMMSS.log` | 개별 실행 아카이브 |

### 로그 형식
```
=== ClaudeScheduler Job Execution ===
Job ID: <uuid>
Job Name: <이름>
Mode: Background
Time: 2026-02-04 10:00:00
======================================
Prompt: <프롬프트 내용>
--- Output ---
<Claude 출력>
--- End of Output ---
Exit Code: 0
End Time: 2026-02-04 10:02:30
```

### 로그 자동 정리
30일이 지난 로그는 자동으로 삭제됩니다.

## 프롬프트 작성 팁

### 기본 형식
```
오늘의 AI 관련 뉴스를 검색하고 요약해서
~/Documents/daily-news.md 파일로 저장해줘.
```

### 복잡한 작업
```
1. WebSearch로 "Claude Code" 관련 최신 뉴스 3개 검색
2. 각 뉴스의 핵심 내용을 3줄로 요약
3. ~/Documents/ai-news/$(date +%Y-%m-%d).md 로 저장
4. 파일 형식: 마크다운, 각 뉴스에 출처 링크 포함
```

> **중요**: `--dangerously-skip-permissions` 플래그가 자동 적용되므로,
> Claude는 파일 읽기/쓰기, 웹 검색, bash 명령 등을 자유롭게 사용합니다.
